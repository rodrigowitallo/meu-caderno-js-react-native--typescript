## 📖 Miniguia de Estudo (Entrega Final)

### 1. Resumos Estruturados
* **JavaScript:** O motor dinâmico. Essencial dominar `async/await` para lidar com dados de APIs.
* **TypeScript:** A camada de segurança. O uso de Interfaces evita que o app quebre em produção por dados inesperados.
* **React Native & Expo:** A união perfeita para criar apps Android/iOS com performance nativa e facilidade de desenvolvimento.

### 2. Glossário
* **Type Safety:** Garantia de que as variáveis seguem o tipo definido.
* **Superset:** Uma linguagem que estende outra (TS estende JS).
* **Hot Reloading:** Recurso do Expo que atualiza o app instantaneamente ao salvar o código.

### 3. Prompts Reutilizáveis
* "Resuma os principais hooks do React Native utilizados para [X]."
* "Refatore o seguinte código JavaScript para TypeScript aplicando interfaces: [COLAR CÓDIGO]."
